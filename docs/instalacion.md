# Instalación local

## Requisitos previos

- Python instalado
- Git instalado
- Entorno virtual configurado
- Dependencias del proyecto instaladas desde `requirements.txt`

## Pasos generales

1. Clonar el repositorio backend.
2. Crear y activar entorno virtual.
3. Instalar dependencias.
4. Configurar variables de entorno.
5. Ejecutar el servidor backend.
6. Validar los endpoints mediante Swagger/OpenAPI.

## Nota

Este entorno corresponde a una ejecución académica/de prueba, no a una implementación productiva formal.
