# Documentación del Backend

Esta sección explica la parte técnica del backend de **All-InOne**.

El backend es el componente encargado de gestionar la lógica del sistema, los datos, la autenticación, los permisos, los módulos y los servicios que consume el frontend.

## Qué cubre esta sección

- Instalación local del backend.
- Arquitectura de monolito modular.
- Organización por rutas, servicios, modelos y esquemas.
- Autenticación con JWT.
- Control de acceso basado en roles y permisos.
- Módulos principales: sitios, plantillas, Blog, Tienda, Auth Público y Analítica.
- Auditoría, soft delete y trazabilidad.
- Pruebas, seguridad, calidad y despliegue de prueba.

!!! note "Alcance"
    Esta documentación corresponde a un entorno académico/de prueba. No representa una guía de operación productiva para una empresa real.
