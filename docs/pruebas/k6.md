# k6

k6 se utiliza para pruebas de rendimiento.

## Qué permite evaluar

- Tiempo de respuesta.
- Comportamiento bajo carga.
- Escenarios de smoke, load, stress o spike.
- Estabilidad del backend bajo solicitudes concurrentes.

## Evidencias relacionadas

- Carpeta `k6/`.
- `docker-compose.k6.yml`.
- Reportes de rendimiento.
