# Seguridad

El backend implementa controles de seguridad orientados a proteger la plataforma SaaS multitenant.

## Controles principales

- Autenticación mediante JWT
- Control de acceso basado en roles y permisos
- Protección de rutas y endpoints
- Aislamiento de información por tenant
- Validación de entradas
- Auditoría de acciones relevantes
- Soft delete para preservar registros importantes

## Consideración

La revisión de seguridad se basa en evidencias disponibles y no representa una prueba de penetración ofensiva formal.
