# Despliegue de prueba

El backend puede contar con evidencias de despliegue técnico en entorno académico o de prueba.

## Herramientas relacionadas

- Docker.
- Jenkins.
- Swagger/OpenAPI.
- k6.
- OWASP ZAP.

## Alcance real

Este despliegue permite validar funcionamiento técnico, pero no equivale a una puesta en producción formal para una empresa.

!!! warning "Diferencia clave"
    Despliegue de prueba no es lo mismo que implementación productiva oficial.
