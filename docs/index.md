# All-InOne Backend

Bienvenido a la documentación técnica del backend de **All-InOne**, plataforma SaaS multitenant desarrollada por **ModularSoft - Grupo 4**.

## Propósito

Este sitio documenta la estructura, arquitectura, instalación, API, pruebas, seguridad y despliegue de prueba del backend.

## Tecnologías principales

- FastAPI
- Python
- SQLAlchemy
- JWT
- RBAC
- Jenkins
- k6
- OWASP ZAP
- SonarCloud / SonarQube
- Snyk

## Estructura general

El backend se organiza mediante una arquitectura de monolito modular, separando responsabilidades por rutas, servicios, modelos, esquemas y módulos funcionales.
