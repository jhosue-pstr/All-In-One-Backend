# Riesgos técnicos de seguridad

## Riesgos principales

- Acceso cruzado entre tenants.
- Rutas protegidas de forma incompleta.
- Roles mal configurados.
- Validaciones insuficientes.
- Errores de servidor expuestos.
- Evidencias de seguridad desactualizadas.

## Mitigación esperada

- Revisar JWT.
- Revisar RBAC.
- Validar endpoints críticos.
- Revisar reportes ZAP/Snyk/Sonar.
- Contrastar documentación con código.
