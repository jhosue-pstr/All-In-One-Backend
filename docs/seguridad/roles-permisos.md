# Roles y permisos

Los roles permiten controlar qué acciones puede realizar cada usuario dentro del sistema.

## Roles principales

- Super Administrador.
- Administrador.
- Editor de Sitios.
- Gestor de Contenido.
- Gestor de Tienda.
- Auditor o Solo Lectura.
- Usuario Público.

## Objetivo

Evitar que usuarios sin autorización puedan crear, modificar, eliminar o visualizar recursos no permitidos.
