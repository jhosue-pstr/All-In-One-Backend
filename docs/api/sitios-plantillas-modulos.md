# Sitios, plantillas y módulos

Estos endpoints sostienen el funcionamiento central de All-InOne.

## Sitios

Permiten administrar tenants o espacios de trabajo de cada negocio.

## Plantillas

Permiten reutilizar diseños o estructuras visuales para crear sitios más rápidamente.

## Módulos

Permiten activar funcionalidades como Blog, Tienda, Auth Público o Analítica según la necesidad del negocio.

## Puntos de auditoría

- Verificar permisos de creación, edición y eliminación.
- Validar asociación correcta con el sitio.
- Revisar activación/desactivación de módulos.
- Comprobar trazabilidad y soft delete cuando corresponda.
