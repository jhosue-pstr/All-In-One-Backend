# Tienda

El módulo Tienda permite gestionar productos y pedidos dentro de un sitio.

## Funciones esperadas

- Productos.
- Categorías.
- Carrito.
- Pedidos.
- Checkout.
- Estados de pedido.
- Imágenes de producto.

## Riesgos principales

- Acceso no autorizado a productos de otro tenant.
- Edición de pedidos sin permisos.
- Falta de validación de datos.
- Inconsistencia entre frontend, backend y documentación.
