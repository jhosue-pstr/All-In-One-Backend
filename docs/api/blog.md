# Blog

El módulo Blog permite gestionar contenido informativo del sitio.

## Funciones esperadas

- Crear publicaciones.
- Editar publicaciones.
- Eliminar o desactivar publicaciones.
- Administrar categorías.
- Subir imágenes.
- Controlar estados de publicación.

## Revisión en auditoría

Se debe validar que las acciones del Blog estén protegidas por permisos y asociadas al sitio correcto.
