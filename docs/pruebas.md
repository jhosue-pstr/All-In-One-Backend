# Pruebas

El backend cuenta con evidencias de pruebas funcionales, rendimiento, seguridad y análisis estático.

## Tipos de pruebas

- Pruebas funcionales
- Pruebas automatizadas
- Pruebas de rendimiento con k6
- Pruebas de seguridad con OWASP ZAP
- Análisis estático con SonarCloud / SonarQube
- Revisión de dependencias con Snyk

## Observación

Las pruebas corresponden a un entorno académico/de prueba.
