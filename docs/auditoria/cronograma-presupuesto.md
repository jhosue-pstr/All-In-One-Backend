# Cronograma y presupuesto

## Cronograma

El cronograma separa dos momentos:

| Periodo | Significado |
|---|---|
| Desarrollo auditado | Periodo en que se desarrolló All-InOne por sprints. |
| Auditoría | Periodo estimado para revisar evidencias y emitir resultados. |

## Presupuesto estimado

El presupuesto de auditoría es referencial y académico. No representa un pago real.

| Recurso | Valor estimado |
|---|---:|
| Auditor líder | S/ 800.00 |
| Auditor técnico | S/ 720.00 |
| Especialista QA | S/ 504.00 |
| Analista documental | S/ 480.00 |
| Herramientas | S/ 100.00 |
| Materiales | S/ 70.00 |
| Imprevistos | S/ 214.00 |
| **Total** | **S/ 2,888.00** |

!!! tip "Cómo explicarlo"
    El presupuesto valoriza el esfuerzo auditor. No significa que la universidad o ModularSoft deban pagar ese monto.
