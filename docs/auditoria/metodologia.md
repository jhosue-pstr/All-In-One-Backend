# Metodología de auditoría

La metodología explica cómo se ejecutará la auditoría.

## Fases

=== "Fase 1"
    **Preparar y planificar**

    - Definir objetivos.
    - Confirmar alcance.
    - Definir criterios.
    - Elaborar Plan de Auditoría.
    - Preparar Checklist SDLC.

=== "Fase 2"
    **Describir el proceso de desarrollo**

    - Revisar informe principal.
    - Revisar Jira.
    - Revisar backend y frontend.
    - Revisar arquitectura, módulos, pruebas y seguridad.
    - Registrar evidencias.

=== "Fase 3"
    **Evaluar y reportar**

    - Comparar evidencias contra criterios.
    - Identificar conformidades y no conformidades.
    - Elaborar matriz de hallazgos.
    - Elaborar informe preliminar y final.

=== "Fase 4"
    **Seguimiento**

    - Presentar resultados.
    - Solicitar plan de acción correctiva.
    - Dar seguimiento a hallazgos.
    - Elaborar acta de cierre.

## Enfoque

La auditoría es evaluativa, no correctiva. Identifica hallazgos y recomendaciones, pero no modifica directamente el sistema.
