## Propósito de esta sección

La auditoría SDLC permite revisar si el desarrollo de All-InOne fue gestionado, documentado, implementado, probado y evaluado de manera coherente.

No se busca certificar oficialmente el sistema ni hacer correcciones directas en el código. El enfoque es **evaluativo, académico y de mejora continua**.

!!! info "Idea principal"
    Esta sección explica cómo se planifica y sustenta la auditoría del ciclo de vida del desarrollo de software de All-InOne.
