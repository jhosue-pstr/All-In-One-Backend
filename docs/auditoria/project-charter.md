# Project Charter de Auditoría SDLC

El **Project Charter** es el documento que formaliza el inicio de la auditoría del ciclo de vida del desarrollo de software de All-InOne.

## Qué define

- Nombre del proyecto de auditoría.
- Sistema auditado.
- Organización auditada.
- Objetivo general.
- Objetivos específicos.
- Alcance y exclusiones.
- Criterios de auditoría.
- Metodología.
- Entregables.
- Cronograma.
- Presupuesto estimado.
- Riesgos.
- Supuestos, restricciones y aprobaciones.

## Por qué es importante

Sin Project Charter, la auditoría podría quedar poco clara. Este documento define qué se revisa, qué no se revisa, con qué criterios se evalúa y qué resultados se esperan.
