# Glosario

| Término | Significado |
|---|---|
| SDLC | Ciclo de Vida del Desarrollo de Software. |
| Project Charter | Documento que formaliza el inicio del proyecto de auditoría. |
| Checklist SDLC | Lista de verificación usada para auditar el proceso. |
| Monolito modular | Arquitectura de una sola aplicación organizada internamente por módulos. |
| SaaS | Software como servicio. |
| Multitenant | Sistema usado por varios tenants o clientes manteniendo datos separados. |
| Tenant | Sitio, cliente o espacio de trabajo dentro de la plataforma. |
| JWT | Token usado para autenticación. |
| RBAC | Control de acceso basado en roles. |
| Soft delete | Eliminación lógica de registros. |
| DevSecOps | Integración de desarrollo, seguridad y operaciones. |
| MkDocs | Herramienta para publicar documentación en forma de sitio web. |
